Mouse Events in JS

onclick()

clicking here will pop an alert.

oncontextmenu()

right click on this element will not open any option menu.

ondblclick()

double clicking will show text telling you that you double clicked.

onmousedown()

Mouse down will set the color of the text to white.

onmouseenter()

taking mouse pointer inside this element will immediately thicken the border.

onmouseleave()

taking mouse pointer outside this will change color of the element

onmouseup()

releasing your mouse on this element will alert you about it.

onmouseover()

everytime you move mouse pointer over this element, it will add random text in the element.
